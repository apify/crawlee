<a name="socialhandles"></a>

Representation of social handles parsed from a HTML page.

The object has the following structure:

```
{
  emails: String[],
  phones: String[],
  phonesUncertain: String[],
  linkedIns: String[],
  twitters: String[],
  instagrams: String[],
  facebooks: String[],
  youtubes: String[],
}
```
